Welcome to the official feedback and support hub for **DocComponentStudio**, the app designed to make document components smarter, faster, and more intuitive.

This repository is your place to:
- Report bugs 🐞  
- Request new features 💡  
- Ask questions ❓  
- Share suggestions ❤️  

Your feedback helps shape the future of DocComponentStudio — thank you for being part of the journey.

---

## 🎨 About DocComponentStudio

**DocComponentStudio** simplifies document creation by turning repetitive tasks into reusable components.  
Whether you're building templates, managing structured content, or automating document workflows, DocComponentStudio gives you the tools to do it elegantly.

---

## 🐞 Report a Bug

If something isn’t working as expected:

1. Go to the **Issues** tab.  
2. Select **New Issue → Bug Report**.  
3. Fill out the details (screenshots encouraged).

Please include:
- What you were doing  
- What you expected to happen  
- What actually happened  
- Steps to reproduce  

---

## 💡 Request a Feature

Have an idea that would make DocComponentStudio better?

1. Open a **Feature Request** issue.  
2. Describe the feature and why it matters.  
3. Add examples or mockups if helpful.

---

## ❓ Ask a Question

Need clarification or guidance?  
Open a **Question** issue — we’re happy to help.

---

## 📸 Attach Screenshots

Visuals help us diagnose issues faster.  
Attach screenshots or short recordings whenever possible.

We use labels to organize feedback:

| Label | Purpose |
|-------|----------|
| 🐞 **bug** | Something isn’t working |
| 💡 **enhancement** | New feature or improvement |
| ❓ **question** | Further information requested |
| ⚙️ **needs‑info** | More details required |
| 🔄 **in‑progress** | Work has started |
